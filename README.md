# 图书管理系统 java197

## 项目概述

图书管理系统是一款基于B/S架构与MVC模式的Web应用程序。该系统实现了对图书信息、借阅信息以及用户信息的全面管理。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**：
  - IDEA
  - JDK 1.8
  - Maven 3.6.1
  - Mysql 8.0
  - Node 14
  - Navicat
  - Redis
- **技术栈**：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

- 普通用户
- 管理员

### 管理员功能

- 图书信息管理（增删改查）
- 图书类别管理（增删改查）
- 借阅信息管理
- 用户管理（增删改查）
- 修改密码

### 用户功能

- 借书
- 还书
- 查看借阅记录
- 修改密码

## 运行环境

系统经过严格测试，确保在以下环境中稳定运行：

- 服务器：符合Java与Node.js运行要求的标准化服务器环境
- 客户端：现代Web浏览器
